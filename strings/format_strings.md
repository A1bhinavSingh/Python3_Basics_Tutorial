
# Format Strings

### Exercise 1

Try the following expressions in a Python shell:

    "{}".format("Hello")
    "{:10}".format("Hello")
    "{:>10}".format("Hello")
    "{1} {0}".format("first", "second")
    "{:5d}".format(42)
    "{:4.1f}".format(3.14159)
    "{:6.3f}".format(3.14159)
